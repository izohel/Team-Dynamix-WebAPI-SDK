Remarks

Currently only supports admin login.
This client abstracts API interaction details, managing authentication tokens, HTTP client configuration, and exposes high-level endpoint request builders. It supports multiple constructors for flexible configuration, including default options, fully custom options, or supplying a preconfigured HttpClient instance.
